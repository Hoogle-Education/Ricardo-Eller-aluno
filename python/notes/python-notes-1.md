## Comando `print( msg )`

O comando `print` recebe um texto e imprime pulando linha.

## Comando `input( msg )`

O comando `input` recebe como parametro opicional uma mensagem a ser exibida antes de solicitar a entrada dados.

## Estruturas de seleção

```py
nota = float(input())

if nota > 7.0 :
  print('Aprovado')
elif nota > 4.0:
  print('Recuperação')
else :
  print('Reprovado')
```
